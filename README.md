# Streaming Web Service

## Overview
유튜브와 유사한 영상 스트리밍 웹 서비스를 1인으로 개발한 프로젝트입니다.

## Role
- 전체 개발 (Frontend + Backend 100%)

## Tech Stack
- JavaScript
- Node.js

## Key Features
- 영상 업로드 및 재생 기능
- 사용자 회원가입 및 로그인
- 댓글 및 콘텐츠 관리 기능

## Contribution

### 1. 서비스 구조 설계
- 클라이언트-서버 구조 설계
- 사용자 요청 처리 흐름 구성

### 2. 백엔드 개발
- API 설계 및 데이터 처리 로직 구현
- 사용자 인증 및 데이터 관리 기능 구현

### 3. 프론트엔드 개발
- 사용자 인터페이스 구성
- 기능별 화면 흐름 설계 및 구현

## Result
- 1인 개발로 웹 서비스 구현 완료
